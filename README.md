# 本次作业实现的内容：（所有作业二的要求均已实现）
1. 实现了开火击中靶子的判定，靶子被击飞的物理效果；
2.实现了手雷功能（瞄准抛物线， 爆炸伤害，物理效果）；
3.主角动作都有（站立，跑动，跳跃，开火，瞄准，换枪等）
4.按下ALT主角可以八方向移动和上下左右开火；
5.主角的枪符合材质要求，加入了手电筒；
6.游戏加入了景深效果。
以下是作业要求外的功能：
7.可以拾起多武器，能够多武器切换；
8.UI跟随当前武器变化；
9.加入了3把不同特色的武器（手枪，步枪，霰弹枪）；
10.武器拥有多种开火模式。
